# Library Management System

## Description
A C# based Library Management System designed to automate and simplify library operations.  
The project applies Object-Oriented Programming (OOP) principles to manage books, members, and borrowing transactions efficiently.

## Features
- Add and manage books.
- Manage library members.
- Borrow and return books.
- Track borrowing records.
- Support different member types.
- Apply Object-Oriented Programming concepts.

## Technologies Used
- C#
- .NET
- Object-Oriented Programming (OOP)
- Visual Studio

## Project Structure
- `Book.cs` - Represents book information.
- `Library.cs` - Handles library operations.
- `Member.cs` - Represents library members.
- `PremiumMember.cs` - Represents premium members.
- `BorrowRecord.cs` - Manages borrowing records.
- `Program.cs` - Application entry point.

## How to Run
1. Clone the repository.
2. Open `Library Management System.sln` using Visual Studio.
3. Build the solution.
4. Run the project.
